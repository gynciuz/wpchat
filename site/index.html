What it does

Orders

List, filter, search, change status, add notes, resend emails — with a confirmation step before anything reaches a customer.

Content

Create posts & pages as drafts with images, then publish on your okay. Edit existing content with preview + confirm.

SEO

Run an SEO / AI-SEO audit and fix titles, meta descriptions, indexing and more through the chat.

Traffic

Ask “how many visitors this week?” — WPChat reads whichever analytics plugin you already have.

Any language

Works in Lithuanian, Russian, Polish and English. Type how you think.

Safe by design

No bulk-destructive actions, no silent deletes. When it can’t do something, it hands you a direct admin link.

Install in 4 steps

  1. Download the latest wpchat-vX.Y.Z.zip from Releases.
  2. Upload it in WP admin → Plugins → Add New → Upload Plugin → Activate.
  3. Add your Anthropic API key from console.anthropic.com in the first-run setup.
  4. Open /wpchat on your site and start typing.

Pricing

Free— open-source (MIT)

You bring your own Anthropic API key and pay Anthropic directly for usage — typically a few cents per request. WPChat collects no payment, so there’s nothing to refund. A hosted WPChat Cloud tier (no key setup) is coming as a ~€12/mo Stripe subscription — join the waitlist.

Privacy

WPChat sends the content of your requests — which can include order and customer data — to Anthropic to generate replies. Your conversation history stays in your own site’s database; your API key never reaches the browser. Optional, PII-free error reporting and an explicit “Report a problem” button are the only data sent to the developer. Full details in PRIVACY.md.
